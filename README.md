# CR-BdE
Site de diffusion des CR du BdE INSA Lyon

## Env variables

```
ENV DATABASE_URL postgres://cr@db/cr
ENV SECRET_KEY ''
ENV DJANGO_ENV 'dev'
ENV MAILGUN_KEY ''
ENV MAILGUN_DOMAIN ''
ENV DEFAULT_FROM_EMAIL ''
ENV ALLOWED_HOSTS "cr.bde-insa-lyon.fr"
ENV GOOGLE_ANALYTICS_PROPERTY_ID ''
```
